# Known limitations v0.8.1

1. Formal checks are bounded executable models, not unbounded theorems.
2. SQLite is a local reference monitor, not a production actuator controller.
3. Distributed partitions, Byzantine participants, and hardware side channels are out of scope.
4. Independent Windows reproduction passed under PowerShell 7.5.8 and Python 3.10.11; this does not establish portability to every Windows environment.
5. The sixth blind model review found no candidate blockers, but it is not human peer review.
6. No external human conceptual reviewer has accepted `v0.8.1`.
7. No external-rater endpoint reliability and validity pilot has been run.
8. Power analysis remains on hold until empirical evaluator variance and final margins exist.
9. No OSF registration or Registered Report has been submitted.
10. No matched-control main-study data have been collected.
11. Quantum and photonic profiles are interface specifications only.
12. Reconciliation is proposal and audit semantics, not a formal merge proof.
13. Privacy, retention, and erasure semantics for a full temporal witness remain open.
14. Passing 190 current tests and 177 regression tests does not establish consciousness, identity, life, personhood, or entity status.
15. Behavioural equivalence can be claimed only for explicitly measured endpoints and conformance gates.
