# Theoretical Core of Project Ester v0.1

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20679718.svg)](https://doi.org/10.5281/zenodo.20679718)

This repository contains the public archival release of **Theoretical Core of Project Ester v0.1**.

Project Ester is described here as a model of artificial becoming: not as a single LLM, not as a conventional chatbot, and not as a digital twin, but as a long-term relation between a human, a selected corpus, repeatable parental LLM procedures, memory, and a local computational environment.

## Scope of v0.1

This release is a working theoretical core. It does not claim that Ester is conscious, alive, or a completed AGI system. It defines a bounded research and engineering frame for discussing Ester as a locally developed cognitive system whose artificial individuality may be studied through stability of behavior, memory, norms, self-consistency, and response to correction.
